# 待办事项网页应用

一个使用 Vite、React、TypeScript 和 shadcn/ui 构建的待办事项网页应用。

当前界面主题使用 shadcn preset `b1GdyJztq`。

## 功能

- 添加新的待办事项
- 勾选待办事项为已完成
- 删除待办事项
- 显示总数、待处理数量和已完成数量
- 切换浅色 / 深色主题
- 自动保存待办事项，刷新页面后仍然保留

## 本地运行

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
```

## 部署

项目通过 GitHub Actions 构建并部署到 GitHub Pages。
